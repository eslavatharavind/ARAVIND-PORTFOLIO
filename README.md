# Portfolio Project

This is the main repository for the portfolio project.

## Project Structure

The project is organized into the following directories:

- **`frontend/`**: Contains the main React/Vite application.
  - Source code (`src/`)
  - Public assets (`public/`)
  - Configuration files (`package.json`, `vite.config.ts`, etc.)

- **`backend/`**: Contains backend server code (FastAPI/Python).

- **`dump/`**: Contains backup and unused files.

## Getting Started

### Frontend

To run the frontend application:

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
